# ✅ 背景图显示问题已修复

## 🔧 问题原因

之前的 CSS 配置中，`#web_bg` 元素缺少 `background-image` 属性，导致无法显示背景图，只显示默认的蓝色渐变。

## ✅ 已修复

在 `source/css/custom.css` 中添加：

```css
#web_bg {
  background-image: url('/img/background.jpg');
  background-attachment: fixed;
  /* ... 其他属性 */
}
```

## 🎨 当前背景效果

现在应该能看到：

1. **背景图显示**：`/img/background.jpg`
2. **缓慢缩放动画**：20秒呼吸式循环（100% → 110%）

4. **毛玻璃卡片**：半透明效果
5. **悬停动画**：卡片上浮效果

## 📝 技术细节

### 修改的文件
- `source/css/custom.css` - 添加背景图 URL

### CSS 规则
```css
#web_bg {
  position: fixed;
  z-index: -999;
  width: 100%;
  height: 100%;
  background-image: url('/img/background.jpg');  /* 关键修复 */
  background-attachment: fixed;                  /* 固定背景 */
  background-position: center;
  background-size: cover;
  background-repeat: no-repeat;
  animation: bgAnimation 20s ease-in-out infinite alternate;
}
```

### 动画效果
```css
@keyframes bgAnimation {
  0% { transform: scale(1); }
  100% { transform: scale(1.1); }
}
```
