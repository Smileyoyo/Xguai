## 🔧 高级配置

### 1. 调整动画速度

打开 `source/css/custom.css`，修改：

```css
/* 当前：20秒循环 */
animation: bgAnimation 20s ease-in-out infinite alternate;

/* 更快：10秒 */
animation: bgAnimation 10s ease-in-out infinite alternate;

/* 更慢：30秒 */
animation: bgAnimation 30s ease-in-out infinite alternate;
```

### 2. 调整缩放幅度

```css
@keyframes bgAnimation {
  0% {
    transform: scale(1);    /* 原始大小 */
  }
  100% {
    transform: scale(1.1);  /* 放大10%，可调整 */
  }
}
```

### 3. 更换不同背景图

```yaml
# _config.butterfly.yml
background:
  image: /img/background.jpg  # 主页背景

# 或者为不同页面设置不同背景
# 需要在 custom.css 中添加特定页面的样式
```
