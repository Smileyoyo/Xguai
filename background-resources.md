## 📥 使用方法

### 静态图片 + CSS 动画（推荐）
```bash
# 1. 下载图片
# 2. 放到 my-blog/source/img/ 目录
# 3. 重命名为 background.jpg
# 4. 运行 hexo clean && hexo g && hexo s
```

### 视频背景（高级）
```bash
# 1. 下载 MP4/WebM 格式视频
# 2. 上传到 CDN 或图床
# 3. 修改配置添加视频背景
```
