前言

欢迎来到我的博客！这是我使用 Hexo + Butterfly 主题搭建的第一篇文章。

为什么搭建博客？

搭建个人博客有很多好处：

  1. 记录学习: 记录学习过程中的知识点和心得体会
  2. 分享经验: 将自己的经验分享给他人，帮助更多的人
  3. 建立个人品牌: 长期坚持写作，可以建立自己的个人品牌
  4. 结交朋友: 通过博客可以结识更多志同道合的朋友

关于本博客

本博客使用了以下技术：

  • Hexo: 一个快速、简洁且高效的博客框架
  • Butterfly: 一款美观、功能丰富的 Hexo 主题
  • GitHub Pages: 免费的静态网站托管服务

后续计划

接下来，我会在博客中分享：

  • 技术学习笔记
  • 项目实战经验
  • 生活随笔
  • 资源分享
