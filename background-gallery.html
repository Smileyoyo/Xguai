📖 使用说明

  1. 点击上面的 "📥 下载此背景" 按钮
  2. 将下载的图片重命名为 background.jpg
  3. 将图片放置到项目目录：my-blog/source/img/background.jpg
  4. 在项目根目录运行命令：
    hexo clean && hexo generate && hexo server
  5. 打开浏览器访问：http://localhost:4000
提示：如果图片加载失败，可能是网络问题。请稍后重试，或直接在浏览器中打开链接下载。
