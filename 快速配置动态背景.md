# 🚀 快速配置 WLOP 动态背景

## 📦 三步快速配置

### 第一步：下载背景图

**推荐 WLOP 鬼刀系列壁纸下载链接：**

1. **Wallhaven (推荐，免费高清)**
   - 蓝色公主：https://wallhaven.cc/full/wq/full/wq6W6w.jpg
   - 粉色公主：https://wallhaven.cc/full/wq/full/wq6W6x.jpg
   - 黑金战士：https://wallhaven.cc/full/wq/full/wq6W6y.jpg

2. **壁纸网站**
   - https://www.dianyabizhi.com/dongman/4046.html (WLOP 鬼刀合集)
   - https://www.3gbizhi.com/wlop/ (WLOP 专题)

3. **直接搜索**
   - Pixiv: https://www.pixiv.net/users/1980643
   - 微博: 搜索 @WLOP

### 第二步：放置图片

```bash
# 将下载的图片放到以下目录
my-blog/source/img/background.jpg

# 或者使用其他文件名，修改配置：
my-blog/source/img/my-background.jpg
```

### 第三步：重新生成

```bash
# 清理缓存
hexo clean

# 生成静态文件
hexo generate

# 启动本地预览
hexo server
```

## 🔧 配置文件说明

当前配置已设置为：

```yaml
# _config.butterfly.yml
background:
  image: /img/background.jpg  # 你的背景图路径
  fixed: true                 # 背景固定
  background_size: cover      # 完全覆盖
```

## 🎯 进阶：使用视频背景

如果想使用视频背景（如 Wallpaper Engine 导出的 MP4）：

### 1. 导出视频
从 Wallpaper Engine 导出 MP4 视频

### 2. 上传到图床
推荐图床：
- imgse.com (免费)
- imgur.com
- GitHub (作为仓库)

### 3. 修改配置
在 `source/css/custom.css` 添加：

```css
#web_bg {
  background: url('你的视频链接.mp4') no-repeat center center;
  background-size: cover;
}

/* 或者使用 video 标签 */
#video-bg {
  position: fixed;
  right: 0;
  bottom: 0;
  min-width: 100%;
  min-height: 100%;
  z-index: -999;
  object-fit: cover;
}
```

## 💡 小贴士

1. **图片优化**
   - 使用 TinyPNG 压缩：https://tinypng.com/
   - 转换为 WebP 格式（体积减少 30-50%）

2. **CDN 加速**
   ```yaml
   # _config.yml
   post_asset_folder: true
   ```

3. **懒加载**
   - 已在配置中启用，图片会在进入视口时加载

4. **移动端优化**
   ```css
   @media (max-width: 768px) {
     #web_bg {
       background-size: cover;
     }
   }
   ```
