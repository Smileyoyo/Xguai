## 🔧 自定义配置

### 修改动画速度
编辑 `source/css/custom.css`：
```css
/* 更快：10秒 */
animation: bgAnimation 10s ease-in-out infinite alternate;

/* 更慢：30秒 */
animation: bgAnimation 30s ease-in-out infinite alternate;
```

### 调整缩放幅度
```css
@keyframes bgAnimation {
  0% { transform: scale(1); }
  100% { transform: scale(1.2); } /* 改为1.2倍 */
}
```

### 更换其他背景图
1. 下载新的背景图
2. 覆盖 `source/img/background.jpg`
3. 运行 `hexo clean && hexo g`

## 💡 常见问题

**Q: 图片加载不出来？**
- 检查文件路径：`source/img/background.jpg`
- 确保文件名是 `background.jpg`（全小写）

**Q: 动画太慢？**
- 修改 `custom.css` 中的 `20s` 为更小的值

**Q: 想要更多背景图？**
- 查看 `background-resources.md`
- 访问 https://pixiv.net/users/1980643 (WLOP官方)
