## 🎯 最常见的错误

### 错误 1：Source 配置错误
**症状**：页面显示 404 或错误
**原因**：Pages Source 选成了 `Deploy from a branch`
**解决**：改为 `GitHub Actions`

### 错误 2：主题文件缺失
**症状**：Actions 日志显示找不到主题
**原因**：主题目录未提交到 Git
**解决**：运行 `git add themes/butterfly/ && git push`

### 错误 3：缓存问题
**症状**：修改后看不到效果
**原因**：浏览器缓存
**解决**：按 `Ctrl + Shift + R` 强制刷新
