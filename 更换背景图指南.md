## 🔄 如何定期更换背景图

### 方法 1：手动更换

1. 下载新图片
2. 替换 `source\img\background.jpg`
3. 运行 `hexo clean && hexo g && hexo d`

### 方法 2：准备多个背景图

在 `source\img\` 目录准备多个背景图：

```
source\img\
  ├── background.jpg      (当前使用)
  ├── background-blue.jpg
  ├── background-pink.jpg
  └── background-dark.jpg
```

更换时只需：
1. 删除 `background.jpg`
2. 将想要的图片重命名为 `background.jpg`
3. 重新部署

### 方法 3：使用 CSS 自动切换（高级）

编辑 `source\css\custom.css`，添加：

```css
/* 每天自动更换背景图 */
#web_bg {
  animation: bgChange 24s infinite;
}

@keyframes bgChange {
  0% { background-image: url(/img/background-blue.jpg); }
  33% { background-image: url(/img/background-pink.jpg); }
  66% { background-image: url(/img/background-dark.jpg); }
  100% { background-image: url(/img/background-blue.jpg); }
}
```

## 🔧 自定义动画效果

### 调整动画速度

编辑 `source\css\custom.css`：

```css
/* 更快：10秒 */
animation: bgAnimation 10s ease-in-out infinite alternate;

/* 更慢：30秒 */
animation: bgAnimation 30s ease-in-out infinite alternate;

/* 当前：20秒 */
animation: bgAnimation 20s ease-in-out infinite alternate;
```

### 调整缩放幅度

```css
@keyframes bgAnimation {
  0% { transform: scale(1); }
  100% {
    /* 改为 1.2 倍（当前是 1.1） */
    transform: scale(1.2);
  }
}
```

### 关闭动画

如果不喜欢动画效果，注释掉：

```css
/*
#web_bg {
  animation: bgAnimation 20s ease-in-out infinite alternate;
}
*/
```

## 💡 常见问题

### Q1: 更换背景图后看不到效果？

**A:** 清除浏览器缓存
- 按 `Ctrl + Shift + R` 强制刷新
- 或等待 1-2 分钟让 GitHub 处理

### Q2: 背景图显示不全？

**A:** 检查图片尺寸
- 确保图片是 1920x1080 或更高
- 使用横向图片（16:9）

### Q3: 想要视频背景？

**A:** 视频背景配置较复杂，建议使用高质量静态图片
- 静态图加载更快
- 更节省流量
- 效果也很好

### Q4: 可以为不同页面设置不同背景吗？

**A:** 可以，但需要额外配置
- 需要修改主题模板
- 或使用页面级别的 CSS
- 建议统一使用一个背景图
