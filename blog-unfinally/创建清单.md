## 创建步骤

### Step 1: 打开飞书开放平台
访问 https://open.feishu.cn/app

### Step 2: 创建应用（重复 7 次）

对每一个应用执行以下操作：

1. 点击「创建企业自建应用」
2. 填写应用信息：
   - **应用名称**：参考上表（如「CEO 助手」）
   - **应用描述**：`虚拟公司多 Agent 协作系统 - [部门名称]`
   - **应用头像**：可上传对应 emoji 图标
3. 点击「创建应用」

### Step 3: 配置每个应用

创建应用后，进入应用详情页完成以下配置：

#### 3.1 添加应用能力
- 点击「添加应用能力」
- 开启「机器人」（必需）
- 开启「网页」（可选）

#### 3.2 权限管理
点击「权限管理」，申请以下权限：

**基础权限（所有应用都需要）：**
- ✅ `im:chat:readonly` - 获取群组信息
- ✅ `im:message.group_msg` - 接收群消息
- ✅ `im:message:send_as_bot` - 以机器人身份发送消息
- ✅ `im:message.group_msg:readonly` - 读取群消息

**可选权限（根据部门职能选择）：**

| 部门 | 额外权限 |
|------|---------|
| CEO | `docx:document` - 文档管理 |
| 产品部 | `docx:document`、`bitable:app` - 文档和多维表格 |
| 开发部 | `docx:document` - 文档 |
| 设计部 | `docx:document`、`drive:drive` - 文档和云空间 |
| 市场部 | `docx:document`、`contact:contact.base:readonly` - 文档和通讯录 |
| 数据部 | `docx:document`、`bitable:app` - 文档和多维表格 |
| 运维部 | `docx:document` - 文档 |

#### 3.3 事件订阅（重要）
- 进入「事件订阅」页面
- 配置方式：长连接
- 订阅事件：添加 `im.message.receive_v1`

#### 3.4 安全设置
- 进入「安全设置」
- 配置「通讯录权限范围」：选择「全部成员」或需要的部门

#### 3.5 版本管理与发布
1. 点击「版本管理与发布」
2. 点击「创建版本」
3. 填写版本信息：
   - 版本号：1.0.0
   - 更新说明：初始版本
4. 点击「保存」→「申请发布」
5. 等待企业管理员审核通过

### Step 4: 收集凭证

每个应用创建完成后，进入「凭证与基础信息」页面，复制：
- **App ID**（如：`cli_xxxxxxxxxxxxxxxx`）
- **App Secret**（需要点击「查看」按钮）

将 7 个应用的凭证填入下方的配置表中。
